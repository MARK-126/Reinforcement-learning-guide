# Libros Recomendados de Reinforcement Learning

## Textos Fundamentales

### 1. Reinforcement Learning: An Introduction
**Autores**: Richard S. Sutton y Andrew G. Barto  
**Edición**: 2nd Edition (2018)  
**Nivel**: Principiante a Avanzado  

**Descripción**: El libro definitivo de RL. Escrito por dos de los fundadores del campo, cubre desde fundamentos hasta métodos modernos.

**Contenido**:
- Parte I: Tabular Solution Methods (Capítulos 1-8)
- Parte II: Approximate Solution Methods (Capítulos 9-13)
- Parte III: Looking Deeper (Capítulos 14-17)

**Dónde conseguirlo**:
- [Versión gratuita online](http://incompleteideas.net/book/the-book-2nd.html)
- [Amazon](https://www.amazon.com/Reinforcement-Learning-Introduction-Adaptive-Computation/dp/0262039249)

**Por qué leerlo**: Es LA referencia estándar. Todo el que trabaja en RL debe leer este libro.

---

### 2. Algorithms for Reinforcement Learning
**Autor**: Csaba Szepesvári  
**Año**: 2010  
**Nivel**: Intermedio a Avanzado  

**Descripción**: Tratamiento matemático riguroso de algoritmos de RL. Más técnico que Sutton & Barto.

**Contenido**:
- Value Prediction
- Control Methods
- Approximate Dynamic Programming
- Convergence Analysis

**Dónde conseguirlo**:
- [Versión gratuita](https://sites.ualberta.ca/~szepesva/papers/RLAlgsInMDPs.pdf)
- Amazon para versión impresa

**Por qué leerlo**: Para entender la teoría profunda detrás de los algoritmos.

---

## Deep Reinforcement Learning

### 3. Deep Reinforcement Learning Hands-On
**Autor**: Maxim Lapan  
**Edición**: 2nd Edition (2020)  
**Nivel**: Intermedio  
**Framework**: PyTorch  

**Descripción**: Enfoque práctico con implementaciones completas. Excelente para aprender a implementar algoritmos.

**Contenido**:
- Fundamentos de RL
- DQN y variantes
- Policy Gradient Methods
- Actor-Critic
- Aplicaciones prácticas

**Dónde conseguirlo**:
- [Packt Publishing](https://www.packtpub.com/product/deep-reinforcement-learning-hands-on-second-edition/9781838826994)
- Amazon, O'Reilly

**Por qué leerlo**: Mejor libro para aprender a implementar DRL desde cero.

---

### 4. Grokking Deep Reinforcement Learning
**Autor**: Miguel Morales  
**Año**: 2020  
**Nivel**: Principiante a Intermedio  

**Descripción**: Explicaciones visuales e intuitivas de conceptos de DRL. Muy didáctico.

**Contenido**:
- Introducción a RL
- Value-based methods
- Policy-based methods
- Model-based RL
- Multi-agent RL

**Dónde conseguirlo**:
- [Manning Publications](https://www.manning.com/books/grokking-deep-reinforcement-learning)

**Por qué leerlo**: Explicaciones muy claras con muchas visualizaciones.

## Programación Dinámica y Control

### 5. Dynamic Programming and Optimal Control
**Autor**: Dimitri P. Bertsekas  
**Edición**: 4th Edition (2017)  
**Nivel**: Avanzado  

**Descripción**: Tratado definitivo sobre programación dinámica. Muy matemático.

**Contenido** (Volumen I):
- Finite Horizon Problems
- Infinite Horizon Problems
- Stochastic Shortest Path Problems

**Contenido** (Volumen II):
- Approximate Dynamic Programming
- Neurodynamic Programming

**Dónde conseguirlo**:
- [Página del autor](http://web.mit.edu/dimitrib/www/dpchapter.html) (algunos capítulos gratis)
- Athena Scientific

**Por qué leerlo**: Perspectiva matemática rigurosa y completa.

## Cómo Leer Estos Libros

### Estrategia de Lectura Efectiva

1. **No leas linealmente**: Salta capítulos según tu interés
2. **Implementa mientras lees**: El conocimiento se asienta con la práctica
3. **Haz los ejercicios**: Los libros de Sutton & Barto y Szepesvári tienen ejercicios excelentes
4. **Toma notas**: Especialmente de las ecuaciones clave
5. **Revisa periódicamente**: Los conceptos se clarifican con relectura

### Plan de Lectura Sugerido (6 meses)

**Mes 1-2**: Sutton & Barto Capítulos 1-6 (Fundamentos y Tabular Methods)
**Mes 3**: Sutton & Barto Capítulos 9-10 (Function Approximation) + Implementar DQN
**Mes 4**: Deep RL Hands-On (Parte I y II) + Implementaciones
**Mes 5**: Sutton & Barto Capítulos 13 (Policy Gradient) + Implementar PPO
**Mes 6**: Leer papers específicos de tu interés + Proyecto propio

## Libros que NO Recomiendo

Algunos libros son obsoletos o de baja calidad. Evita:
- Libros anteriores a 2015 (excepto clásicos)
- Libros que prometen "dominar RL en X días"
- Libros sin matemáticas (RL requiere matemáticas)

---

**Recomendación final**: Empieza con Sutton & Barto. Es gratis, completo y es el estándar. Complementa con implementaciones prácticas usando Deep RL Hands-On.
